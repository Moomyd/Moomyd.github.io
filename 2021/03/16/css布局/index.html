css布局

发表于 2021-03-16 | 分类于 css

浮动

能将一个元素拉到容器的一侧，这样文档流就能够包围它。
浮动本身不是为了实现布局而设计的。

学习浮动的必要性：

  • 要支持IE浏览器，只有IE10和IE11支持Flexbox，还有一些bug
  • 为了维护旧代码
  • 浮动布局不需要很多标记
  • 要求将图片移动到网页一侧
    双容器模式：将内容放到两个嵌套的容器中，内层容器设置外边距，让它在外层容器居中。

    容器折叠和清楚浮动

    容器折叠：浮动元素不同于不同文档流的元素，他们的高度不会加到父元素上

清除浮动：clear:both 在html结尾中添加空标记，使用css清除浮动

使用伪元素（：before ：after 用来在元素的开始或者结束位置插入内容)
使用display:table可以包含外边距

浮动陷阱

浏览器会将浮动元素尽可能的放在靠上的位置。可以使用nth-child去清除每行第一个浮动

媒体对象和BFC

让图片在一侧，一段文字出现在图片的旁边叫做媒体对象

媒体对象里面的文字默认围绕浮动图片，为了文字不和浮动图片重叠，建立块级格式化上下文(BFC)

BFC是网页的一块区域，元素基于这块区域布局，它将内部与外部上下文分隔开：

  • 包含内部所有元素的上下外边距不会产生外边距折叠
  • 包含内部所有的浮动元素
  • 不会和BFC外面的浮动元素重叠

创建BFC：(通常设overflow)

  • float:left/right
  • overflow:hidden/auto/scroll
  • display:inline-block/table-cell/flex等
  • position：absolute/fixed

    网格系统

    构建网格系统可以提高代码的复用率。

    Flexbox

    为元素添加display:flex该元素变成弹性元素，直接子元素变成弹性子元素

属性：

  • flex-direction 指定主轴方向 row row-reverse column column-reverse
  • flex-wrap 指定了弹性子元素是否在弹性容器内折行显示
  • justify-content 控制子元素在主轴上的位置
  • align-items 控制副轴上的位置
  • align-content 如果开启了flex-wrap会控制子元素在副轴上的间距
  • flex-grow 增长因子，决定子元素在主轴方向扩展的大小
  • flex-shrink 收缩因子
  • flex-basis 子元素初始大小
  • align-self 在副轴上的对齐方向
  • Order 整数

    网格布局

    Flexbox和网格布局：
  • Flexbox本质上是一维的而网格是二维的
  • Flexbox是以内容为切入点由内向外工作，而网格是从外向内
  • flex换行后没法让上一行和下一行对齐，网格可以对齐
